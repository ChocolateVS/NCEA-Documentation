Plotting Complex Number on argand diagram

Every complex number can be expressed as a point in the complex plane as it is expressed in the form a+bi where a and b are real numbers.
a describes the real portion of the number and b describes the complex portion.
By using the x axis as the real number line and the y axis as the imaginary number line you can plot the value as you would (x, y)

For example, plot z = 3 + 2i
